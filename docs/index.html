Creating content

Writing Content Every blog and website needs to have some content, so let’s cover how to do that. Inside your Hugo folder, run hugo new posts/hello-world.md. This’ll create a new folder called posts in your content directory, and put a file called hello-world.md in it. If you’re not familiar with Markdown, it’s a markup language that’s commonly used to write and create rich content, and it’s the language that Hugo uses.

This takes all the files and puts them through Hugo to churn out the HTML and CSS that we need. This building is why we call Hugo a static site generator - it generates static sites which can go on a CDN instead of a more costly server.
